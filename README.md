# govhack-2016

## installation

    $ npm install
    $ npm setup

## build tasks

    $ npm run build
    $ npm run build:prod
    $ npm run js
    $ npm run js:watch
    $ npm run js:prod
    $ npm run css
    $ npm run css:watch
    $ npm run css:prod
